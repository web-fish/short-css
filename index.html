Универсальный CSS селектор *

стилизует все элементы на странице

.parent * {
              box-sizing: border-box;
            
}
            <div class="parent"
.child
              </div>

Контекстный селектор

в блоке с классом parent стилизует элемент с классом child через пробел

.parent .child {
              background-color: yellowgreen;
            
}
            <div class="parent"
.child
              </div>

Комбинация селекторов

выберет элемент по тегу div внутри блока parent

              </div>

Еще про комбинацию селекторов

И еще про комбинацию селекторов

Соседний селектор +

позволяет выбрать самый первый элемент, который идет сразу после указанного

.child + div {
              background-color: blueviolet;
            
}
            <div class="parent"
.child (указанный элемент)
.div
              </div>

Селектор прямого потомка >

позволяет выбрать все элементы, которые являются прямыми потомками
указанного блока, на первом уровне вложенности 

.parent > tag {
              background-color: darkcyan;
            
}
.parent > * {
              background-color: darkcyan;
               
}
            <div class="parent"
tag
tag
              </div>

Селектор комбинатор ~

выберет все элементы после указанного (кроме первого — все, кроме указанного), расположенные на первом уровне вложенности. Т.е внутри блока .parent, есть элемент .child И после него него выбираем все елементы с каким-либо тэгом. 

.parent .child ~ tag {
              background-color: orangered;
            
}
            <div class="parent"
.child
tag
              </div>

CSS селекторы :not(selector) и :nth-child()

Псевдокласс отрицания, если вам нужно выбрать все элементы кроме тех, которые соответствую указанному условию 

.parent > :not(:nth-child(1)) {
              background-color: lemonchiffon;
            
}
.parent :nth-child(1) {
              background-color: chartreuse;
               
}
.child:last-child {
                background-color: cornflowerblue;
                 
}
            <div class="parent"

              </div>

Селекторы атрубутов

выберет все элементы, у которых присутствует указанный атрибут

background-color: red;
}

background-color: red;
}
input[type="text"] {
background-color: red;
}
span[name="value"] {
background-color: red;
}
img[src$=".jpg"] {
background-color: red;
}
a[data-link="value"] {
background-color: red;
}
