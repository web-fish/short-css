Универсальный CSS селектор *

стилизует все элементы на странице

<html>
 <head></head>
 <body></body>
</html>

* {
 padding: 0;
 margin: 0;
 border: 0;
 box-sizing: border-box;
}

Соседний селектор +

позволяет выбрать самый первый элемент, который идет сразу после указанного 

<div class="parent">
 <div class="child"></div>
 <div></div>
 <div></div>
</div>

.child + div {
 background-color: #73e16c;
}

<div class="child"></div>
<div class="child"></div>
<div class="child"></div>

Селектор прямого потомка >

позволяет выбрать все элементы, которые являются прямыми потомками указанного блока, на первом уровне вложенности 

<div class="parent">
 <div></div>
 <div></div>
</div>

.parent > div {
 background-color: #73e16c;
}

<div class="child"></div>
<div class="child"></div>

Селектор комбинатор ~

выберет все элементы после указанного (кроме первого — все, кроме указанного). Т.е внутри блока .parent, есть элемент .child и после него него выбираем все элементы по тэгу. 

<div class="parent">
 <div class="child"></div>
 <div></div>
 <input type="text">
 <div></div>
</div>

.child ~ div {
 background-color: #73e16c;
}

<div class="child"></div>
<div></div>
<input type="text"> 
<div></div>

Контекстный селектор

в блоке с классом parent стилизует элемент с классом child через пробел 

<div class="parent">
 <div class="child"></div>
</div>

.parent .child {
 background-color: #73e16c;
}

<div class="child"></div>

Селектор атрибутов

выберет все элементы, у которых присутствует указанный атрибут 

<div class="parent">
 <a href="javascript:void(0);"></a>
 <input type="text">
 <img src="" alt="">
</div>

a:link {
 text-decoration: none;
}
input[type="text"] {
 font-family: inherit;
}
img[src$=".jpg"] {
 max-width: 100%;
}

<a href=""></a>
<input type="text">
<img src="" alt="">

Комбинация селекторов

блок .parent имеет класс .active

<div class="parent">
</div>

.parent.active {
 background-color: #73e16c;
}

выберет элемент(ы) по тегу div внутри блока parent

 <div></div>
 <div></div>
</div>

.parent div {
 background-color: #73e16c;
}

Псевдокласс отрицания

если вам нужно выбрать все элементы кроме тех, которые соответствую указанному условию 

<div class="parent">
 <div></div>
 <div></div>
</div>

.parent > :not(:nth-child(1)) {
 background-color: #73e16c;
}

<div></div>
<div></div>

Псевдокласс :nth-child()

находит один или более элементов, основываясь на их позиции среди группы соседних элементов 

<div class="parent">
 <div class="child"></div>
 <div class="child"></div>
</div>

.child :nth-child(2) {
 background-color: #73e16c;
}

<div class="child"></div>
<div class="child"></div>

Псевдокласс :checked

выбирает только выбранные чекбоксы и радио кнопки 

<div class="parent">
 <input type="checkbox" id="checkbox">
 <label for="checkbox">checkbox</label>
</div>

input[type="checkbox"]:checked + label {
 background-color: #73e16c;
}
